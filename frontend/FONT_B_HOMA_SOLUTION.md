# راه حل مشکل فونت B Homa

## مشکل:
فونت B Homa در Google Fonts با URL فعلی در دسترس نیست (خطای 404).

## راه حل‌های موجود:

### راه حل 1: دانلود فونت و قرار دادن به صورت محلی (پیشنهادی)

1. فونت B Homa را از یکی از منابع زیر دانلود کنید:
   - از سایت رسمی فونت‌های فارسی
   - یا از Google Fonts با استفاده از Google Fonts Helper

2. فایل فونت (فرمت woff2) را در مسیر زیر قرار دهید:
   ```
   frontend/public/fonts/B-Homa.woff2
   ```

3. کد به صورت خودکار از فایل محلی استفاده می‌کند.

### راه حل 2: استفاده از فونت جایگزین (سریع‌ترین)

اگر نمی‌توانید فونت B Homa را پیدا کنید، می‌توانید از Vazirmatn استفاده کنید که در Google Fonts موجود است و قبلاً در پروژه استفاده شده است.

برای این کار، در فایل `InvoiceImageHelper.tsx` خطوط زیر را پیدا کنید و `'B Homa'` را به `'Vazirmatn'` تغییر دهید:

```typescript
font-family: 'B Homa', 'Vazirmatn', 'Tahoma', sans-serif !important;
```

به این شکل:

```typescript
font-family: 'Vazirmatn', 'Tahoma', sans-serif !important;
```

### راه حل 3: استفاده از CDN دیگر

اگر می‌دانید فونت B Homa در CDN دیگری موجود است، می‌توانید URL آن را در کد قرار دهید.

---

## توصیه:
بهترین راه حل، استفاده از فایل محلی است (راه حل 1) چون:
- سریع‌تر است
- وابستگی به اینترنت ندارد
- با html2canvas بهتر کار می‌کند
- قابل اعتمادتر است
